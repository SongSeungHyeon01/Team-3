# PRD v2 — 사내 지식관리 플랫폼

## 한 단락 PRD
사내 보안 규정상 외부 API가 차단된 로컬 폐쇄망 환경에서, HWP/PPT/스캔 PDF로 파편화된
NAS 자료와 파일명 규칙 부재로 자료 탐색에 최대 30분을 허비하는
R&D 중소기업 입사 3개월 차 신입사원을 위한 사내 지식관리 플랫폼(KM Platform) MVP.

## 핵심 기능 (MVP)
1. 포맷 무관 문맥 검색 (PyMuPDF + EasyOCR + BM25 + turbovec)
2. 최신 표준 문서 식별 및 버전 관리
3. 검색 결과 출처 표시 (파일명·경로·페이지·하이라이트)
4. 파일·폴더 업로드 및 원본 구조 유지

## 성공 지표
- 원하는 문서 상위 5개 이내 포함률 >= 80%
- 자료 탐색 시간 30분 -> 5분 이내
